# 早弁チャレンジ
  ニコニコ生放送内で遊べるニコ生ゲームです。Akashic Engine を使用しています。  

## 遊び方
  先生の目を盗んで早弁だ！見つかったら説教されるぞ！  
  画面をクリック中は早弁します。時々先生が振り返るので気をつけよう！ 

## 基本テクニック
  * 先生が生徒側に振り返る直前には必ずメガネがズレます。見逃すな！でもまれにフェイクもあるぞ！
  * 先生が黒板側に振り返るとき、まれにフェイントします。要注意だ！
